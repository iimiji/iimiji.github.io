Pillpop

A Smart Medication Management App. Designed to enhance medication adherence through real-time tracking, personalized reminders, and an intuitive UI.

Tools Used: React Native, Firebase, Expo, Google Maps API, AsyncStorage, Figma
Role: UX/UI Design, Full-Stack Development, Deployment

View Code

Project Overview

Pillpop is a mobile application designed to assist users in managing their daily medication intake efficiently. The app integrates real-time tracking, personalized reminders, and an intuitive UI to enhance medication adherence. The project was entirely conceptualized, designed, and developed by Miji Kim, covering research, UI/UX design, development, and deployment.

This application was developed to address a crucial issue in healthcare: medication non-adherence. Many individuals, especially those managing chronic conditions, struggle to take their medications as prescribed. Pillpop provides a seamless, engaging, and effective way to help users maintain their medication schedules, ultimately contributing to better health outcomes.

Role & Contributions

Concept Development & Market Research

The project began with an in-depth analysis of existing medication management solutions. Through market research and competitor analysis, I identified gaps in user experience, such as complex navigation, lack of intuitive reminders, and insufficient tracking mechanisms. This research informed the core functionalities of Pillpop, ensuring that it provided a streamlined and effective solution for medication tracking.

UI/UX Design

Using Figma, I developed low-fidelity wireframes to outline the app’s core workflows and ensure a logical, user-friendly experience. High-fidelity prototypes were then created, focusing on minimalist aesthetics, ease of use, and accessibility. Design decisions were based on user feedback, ensuring an intuitive and visually appealing interface.

Full-Stack Development

I built the application using React Native for cross-platform support, ensuring smooth performance on both iOS and Android. Firebase Firestore was integrated to enable real-time medication tracking, while Google Maps API allowed users to locate nearby pharmacies. Additionally, AsyncStorage was used to store medication images locally, reducing dependency on cloud storage.

User Testing & Iteration

To refine the user experience, I conducted usability tests with potential users. Feedback revealed areas for improvement, such as simplifying medication entry, enhancing reminder alerts, and refining UI elements. These insights led to iterative updates, improving the app’s overall effectiveness.

Final Deployment & Documentation

After finalizing the core features, I documented the development process, API integrations, and design principles to facilitate future updates and potential scaling.

Problem Statement

Medication non-adherence is a significant issue worldwide, with nearly 50% of patients failing to take prescribed medications correctly. Many factors contribute to this, including forgetfulness, confusion over dosages, and lack of engaging tracking methods.

Key User Pain Points

  • Forgetting to take medications at the right time.
  • Difficulty in keeping track of multiple medications.
  • Lack of engaging and interactive feedback to encourage adherence.
  • Limited access to nearby pharmacies when prescriptions need refilling.

Market Research & Insights

  • The global digital health market is expanding rapidly, with an increasing demand for personalized healthcare solutions.
  • User surveys revealed that 60% of respondents found existing medication apps too complex, leading to non-adherence.
  • Research indicated that audio and visual reminders significantly improve medication adherence rates.

Solution & Key Features

Pillpop was developed to address these challenges with a focus on simplicity, engagement, and effectiveness. The app provides users with an interactive way to track medications, set reminders, and find pharmacies easily.

1. Daily Medication Tracking

  • Users can add, edit, and delete medications effortlessly.
  • A progress tracker allows users to log medications as "taken," reinforcing adherence through a visual indicator.
  • The completion of daily medication triggers celebratory animations and sounds to positively reinforce behavior.

2. Smart Reminders & Audio Feedback

  • Customizable alarm notifications ensure users never miss a dose.
  • Audio reminders, implemented with Expo-av, provide an interactive experience to alert users effectively.
  • A blinking animation feature visually reinforces the importance of adherence.

3. Nearby Pharmacy Search

  • Using Google Maps API, the app enables users to find nearby pharmacies with ease.
  • Real-time location tracking ensures the most relevant search results, displaying distance and operating hours.

4. Medication Information Retrieval

  • Integration with FDA’s API allows users to search for detailed medication information within the app.
  • This feature provides safety information and reduces the need for external research.

5. Local Storage for Medication Photos

  • Users can capture and store images of their medications using the built-in camera functionality.
  • AsyncStorage is utilized for local storage, reducing cloud costs while ensuring seamless access.

6. Interactive Progress & Gamification

  • React Native Animated API is used to create dynamic progress tracking visuals.
  • This gamification approach enhances motivation, making medication adherence more engaging and rewarding.

7. Medical News Feed

  • Live medical news updates sourced via external health news API.
  • Users can browse trending articles related to healthcare and medication.
  • Clicking on articles opens the full content in an in-app browser, ensuring seamless access to relevant information.

Challenges & Solutions

High Firebase Storage Costs

Implemented local storage via AsyncStorage to minimize cloud dependency and reduce operational expenses.

Slow API Response for Medication Searches

Optimized API response caching to ensure faster load times and improved data retrieval efficiency.

Users Ignoring Notifications

Introduced audio reminders and animated alerts to capture attention and increase user engagement.

Background Notifications Not Triggering

Integrated React Native Background Task API to ensure notifications are delivered reliably even when the app is running in the background.

Final Outcome

  • Successfully developed a fully functional cross-platform mobile application.
  • Improved medication adherence rates through interactive feedback and real-time tracking.
  • Integrated real-world APIs for pharmacy search and medication information retrieval.
  • Reduced Firebase costs with an optimized local storage solution.
  • Received positive user feedback, confirming the app’s ease of use and effectiveness.

Key Takeaways & Reflection

Developing Pillpop was an insightful journey that strengthened my expertise in UX/UI design, mobile development, and healthcare technology. Throughout the process, I tackled real-world challenges such as API integrations, optimizing local storage, and improving user engagement strategies.

By combining human-centered design, real-time tracking, and gamification, I was able to create a meaningful solution that enhances medication adherence. This project further solidified my passion for Human-Computer Interaction (HCI) and digital health innovations, and I look forward to exploring more ways technology can positively impact healthcare.
